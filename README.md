### 🌐 [Currículum Vitae Digital](https://pablochacon08.github.io/curriculum/)

<img align="right" src="https://user-images.githubusercontent.com/74038190/212257468-1e9a91f1-b626-4baa-b15d-5c385dfa7ed2.gif" width="160"/>

> Sitio web personal construido desde cero como portafolio profesional online.

Incluye secciones de experiencia, formación, proyectos, referencias y formulario de contacto. Layout de dos columnas con **CSS Grid**, navegación sticky con **Flexbox**, animaciones `fadeInUp` y diseño completamente responsivo.

*Conceptos clave aplicados:*

1. Layout responsivo con **CSS Grid** de dos columnas
2. Navegación sticky con **Flexbox**
3. Animaciones de entrada `fadeInUp` con CSS puro
4. Variables CSS para paleta de colores consistente
5. Media queries para adaptación móvil
